Appel de l'API

Pour constituer les URI d'appel à l'API vous devez faire précéder les Path indiqués ci-dessous par : https://www.epiceventscrm.com

→ Exemple : https://www.epiceventscrm.com/api/token/

Authentification

L'authentification se fait via le standard JWT (JSON Web Token).
